# ООП-подход для создания своих сервисов

## Описание 
Проект по созданию и инициализированию классов
Основной можуль находится в директории src/product.py. Также имеются тесты в соответствующих директориях и файд README.md
есть файл .gitignore и .flake8 с конфигурациями лучших практик
## Тестирование
Написал тесты в директории tests/test_product.py
Покрытие кода тестами составляет 100%. Для выяснения покрытия необходимо ввести команду pytest --cov
## Установка 
```git clone https://github.com/BronnAlex/introduction_in_oop_classes```
## Использование
Подготовка кода для того, чтобы появилось ядров интернет магазина
В дальнейшем для этого ядра возможно будет реализовать любой интерфейс — от сайта до телеграм-бота.
