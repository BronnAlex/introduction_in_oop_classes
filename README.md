# ООП-подход для создания своих сервисов

## Описание 
Проект по созданию и инициализированию классов
Основной модуль находится в директории src/product.py. Также имеются тесты в соответствующих директориях и файд README.md
есть файл .gitignore и .flake8 с конфигурациями лучших практик
Добавил и расширил функционал ранее созданных классов, использовал геттеры и сеттеры, а также классметод
## Тестирование
Написал тесты в директории tests/test_product.py
Написал тесты для нового функционала, подправил прошлые тесты, так как изменились слегка данные с учетом приватных атрибутов
Покрытие кода тестами составляет 98%%. Для выяснения покрытия необходимо ввести команду pytest --cov
## Установка 
```git clone https://github.com/BronnAlex/introduction_in_oop_classes```
## Использование
Подготовка кода для того, чтобы появилось ядров интернет магазина
В дальнейшем для этого ядра возможно будет реализовать любой интерфейс — от сайта до телеграм-бота.
